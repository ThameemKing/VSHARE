# VShare - Food Donation App

A Flutter-based mobile application that connects people who have extra food with those in need. 
The aim of this app is to reduce food waste and make food sharing simple and effective.

## Features

- User registration and login
- Donor and receiver roles
- Add and view food donations
- Browse nearby donations
- Accept donation requests
- Update donation status
- Google Maps integration
- Easy-to-use interface

## Tech Stack

- Flutter (Dart)
- Firebase (Authentication, Firestore, Storage)
- Google Maps API
